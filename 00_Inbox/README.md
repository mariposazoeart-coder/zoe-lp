# 00_Inbox（何でも投げ込みボックス）

ここには **何でも投げ込んでOK** です。

- お客様の声（メール・DM・アンケートの文面など）
- セッションや面談の文字起こし
- ふとしたメモ、アイデア、あとで整理したいこと

## 使い方

1. とりあえずここに新規ノートとして置く（形式は気にしなくてOK）
2. あとで `/feedback-sort` や「Inbox整理して」で、`20_Knowledge/Feedback/` などへ振り分けます
3. 振り分け（移動）の前には必ずプランを提示するので、確認してから移動します

> 迷ったらまずここ。整理は後回しで大丈夫です。
